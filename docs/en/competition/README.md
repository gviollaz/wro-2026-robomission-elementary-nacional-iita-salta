# Competition - WRO RoboMission Elementary 2026

See Spanish version for full details: `docs/es/competition/README.md`

- **Theme:** Robots Meet Culture
- **Challenge:** Robot Rockstars (music festival)
- **Age:** 8-12 | **Max score:** 255 pts | **Max run time:** 2 min

## Missions
1. Connect cables to amplifier (30 pts)
2. Prepare the show - microphone + instruments (65 pts)
3. Play the song - 6 colored notes to staff (120 pts)
4. Bonus - don't damage stage objects (40 pts)

Randomization: 4 of 6 notes randomized each round -> color sensor required.
